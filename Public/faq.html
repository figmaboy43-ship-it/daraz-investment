Frequently Asked Questions

১. Daraz Investment কি?

এটি একটি ডেমো ইনভেস্টমেন্ট সাইট, যেখানে Supabase ব্যবহার করে Authentication, Database, Storage, Dashboard ইত্যাদি দেখানো হয়েছে।

২. এখানে প্যাকেজ কেনা কি সত্যি কাজ করে?

এটি বাস্তব ইনভেস্টমেন্ট নয়। সব পেমেন্ট, প্রুফ আপলোড, রিটার্ন — সবই শিক্ষামূলক উদ্দেশ্যে।

৩. পেমেন্ট কিভাবে কাজ করে?

User → প্যাকেজ Buy → TX Proof Upload → Admin Verify → Dashboard এ দেখাবে। (Supabase Storage + Database দিয়ে তৈরি)

৪. ডেটা কি নিরাপদ?

হ্যাঁ, Supabase RLS Policies ব্যবহার করা হয়েছে। বাস্তব সাইটে আরও শক্ত নিরাপত্তা যোগ করতে হবে।

৫. Dashboard কী?

Dashboard হলো আপনার ব্যালান্স, প্যাকেজ হিস্ট্রি, উইথড্র রিকোয়েস্ট দেখার জায়গা।

৬. Admin Panel কী করে?

Admin পেমেন্ট প্রুফ যাচাই, প্যাকেজ তৈরি/এডিট, উইথড্র রিকোয়েস্ট approve করতে পারে। (এই ডেমোতে basic admin page আছে)

৭. এই সাইটকে কি নিজের কাজে ব্যবহার করতে পারি?

অবশ্যই। শুধু বাস্তব ইনভেস্টমেন্ট প্ল্যাটফর্ম বানালে আইনগত পরামর্শ নিতে হবে।
